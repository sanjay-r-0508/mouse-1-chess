# 📖 Overview

Advanced Chess AI is a browser-based chess application featuring a responsive chessboard, intelligent AI opponent, move validation, game-state tracking, and multiple difficulty levels.

The game utilizes the Chess.js library for chess rules and move validation while implementing a custom AI using the Minimax Algorithm with Alpha-Beta Pruning for strategic gameplay.

# 🤖 AI Decision Making

The AI uses:

## Minimax Algorithm

The Minimax algorithm evaluates future board positions and selects the move that maximizes its advantage while minimizing the opponent's advantage.

---

## Alpha-Beta Pruning

Alpha-Beta pruning improves performance by:

- Reducing unnecessary calculations
- Eliminating weak move branches
- Increasing AI response speed
